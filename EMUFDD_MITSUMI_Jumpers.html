MITSUMI Jumpering


D509V MITSUMI drives






 DS0 

 Drive select 0 

 DS1 

 Drive select 1 

 DS2 

 Drive select 2 

 DS3 

 Drive select 3 

 MS 

 Drive Select controls motor on 

 MM 

 Motor ON controls motor on 

 RDY 

 Ready output on Pin 34 

 DC 

 Disk Change output on Pin 34 

 LD 

 Low Density output 

 HD 

 High Density output 

 2S 

 Dual speed (300 rpm/360 rpm) 

 HS 

 High speed (360 rpm)

 DS0 

 Drive select 0 

 DS1 

 Drive select 1 

 DS2 

 Drive select 2 

 HI 

 High Density input 

 LI 

 Low Density input 

 SW 

 Shift from Normal Density to High Density through disk insertion 

 LO 

 Low Density output 

 HO 

 High Density output
