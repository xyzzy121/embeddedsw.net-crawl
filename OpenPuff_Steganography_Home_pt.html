OpenPuff - Yet not another steganography SW




Portuguese introduction by users & Internet reviews.


OpenPuff é uma ferramenta profissional para a esteganografia:


  • HW gerador de números aleatórios (CSPRNG)

  • Esteganografia negável

  • Cadeia de transportadoras (até 256MB de dados ocultos)

  • Seleção do nível de bits das transportadoras

  • Moderna Multi-Criptografia (16 algoritmos)

  • Níveis de ofuscação de dados ocultos (3 senhas)

  • Resistência à esteganalisis X-quadrado (de primeira ordem)


Níveis de segurança e ofuscação unicos:


  • Criptografia de chave simétrica 256+256 bits com extensão de senha KDF4

  • Scrambling dos dados a chave simétrica de 256 bits (mistura em base CSPRNG)

  • Whitening dos dados (mistura com ruído branco em base CSPRNG)

  • Codificação de bits das transportadoras adaptável e não-linear

  

OpenPuff suporta vários formatos de transportadoras:


  • Arquivos de imagem (BMP, JPG, PCX, PNG, TGA)

  • Arquivos de áudio (AIFF, MP3, NEXT/SUN, WAV)

  • Arquivos de vídeo (3GP, MP4, MPG, VOB)

  • Arquivos de Flash-Adobe (FLV, SWF, PDF)


OpenPuff é um software portátil/oculto:


  • Estrutura portátil nativa (sem instalação, sem chaves de registro e arquivos .ini)

  • Compatível com a prevenção de execução de dados (DEP on)

  • Suporte multi-processo (multithread, até 32 CPUs) = processamento mais rápido


OpenPuff é seguro:


  • Livre de spyware / adware

  • Totalmente redistribuível

  • Cripto-biblioteca principal de Criptografia em código aberto (libObfuscate)
