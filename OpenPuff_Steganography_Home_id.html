OpenPuff - Yet not another steganography SW




Indonesian introduction by users & Internet reviews.


Jika kamu tidak ingin file atau dokumen pribadimu dibuka atau dilihat orang lain, apa yang akan kamu lakukan? Apakah kamu mewinrarnya kemudian kamu beri password untuk membukanya? Atau kamu menghiddennya? Semua cara yang kamu lakukan itu sudah termasuk cara yang kuno.


OpenPuff adalah software steganografi yang sangat kuat sehingga memungkinkan kamu untuk menyembunyikan file atau dokumen rahasiamu dari orang lain.


Software ini bekerja dengan menyembunyikan file rahasiamu ke dalam file lain. Jadi kamu menyembunyikan file didalam file lain. OpenPuff dapat menyembunyikan file yang berformat BMP, JPG, PNG, MP3, WAV, MP4, MPG, FLV, SWF, PDF dan masih banyak lagi.
