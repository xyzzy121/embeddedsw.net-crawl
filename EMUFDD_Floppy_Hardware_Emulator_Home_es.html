IDE EMUHDD

[ SCSI Manual IDE Manual ] EMUHDD ("Emulator of Hard Disk Drive") es una tecnología combinada que permite emular cualquier tipo de disco duro o magneto óptico o unidad extraíble. La comunicación con la maquinaria soporta una amplia gama de protocolos, ATA-IDE, PATA-IDE, ULTRA-SCSI, SCSI I, SCSI II. Los datos se almacenan en una selección de soportes, CF o SDC moderno, y son siempre fácilmente accesibles en la PC para una rápida copia de seguridad y restauración. Cada EMUHDD está configurado a medida para coincidir exactamente con la geometría de la unidad original (nombre, tamaño, pistas, sectores...). No hay límite en el sistema operativo compatible con la emulación: Win/Dos/Unix y cualquier otro sistema operativo antiguo. cada aplicación también recibirá una guía de mantenimiento preventivo recomendado, cada 1/5 años, garantizando una tasa de fracaso única de 0% en el tiempo.


Network EMUFDD

[ NET Manual ] EMUFDD ("Emulator of Floppy Disk Drive") es una tecnología específica para emular cualquier tipo de unidades de disquete, 8"/5,25"/3.5". Los datos se almacenan en una FRAM interna que se comporta como un disco duro, ninguna pérdida de datos durante el apagado. Redes de TCP/IP robustas, 80x más rá que un serial RS232 y comunicación de grado industrial con un número ilimitado de PC distribuidos. Interconexión robusta de emulación y redes y retransmisión aseguran el intercambio fiable de datos en cualquier circunstancia. Soporte de scripting avanzado se incluye para una perfecta integración DNC. En caso de "inactividad de la red", un puerto USB adicional y completamente funcional está también disponible. Ningún mantenimiento preventivo es necesario debido a la garantía única de 20 años de retención de datos.


USB EMUFDD

[ USB Manual ] Basado en la misma tecnología de el EMUFDD de redes, la versión de sólo USB es adecuado para usuarios que no tienen acceso a una conexión de red o que demandan una baja cantidad de datos por uso diario. La misma FRAM interna, que se comporta como un disco duro, puede ser cargada/guardada utilizando una llave USB estándar. No hay necesidad de formatar la llave USB con partición oculta especial o de mantener la llave USB siempre insertada después de el intercambio de datos. Se puede utilizar la pantalla numerada 00-99 para elegir los subdirectorios y las imágenes no estándar, ambos son soportados.
